🔗 Connexion Google Sheets

Pour connecter ton Google Sheet, tu as besoin de l'ID du sheet et d'une clé API Google.

Étape 1 — Trouve l'ID du sheet : c'est la partie de l'URL entre /d/ et /edit.
Ex : https://docs.google.com/spreadsheets/d/TON_ID_ICI/edit

Étape 2 — Crée une clé API : va sur console.cloud.google.com → Identifiants → Créer une clé API → puis active "Google Sheets API".

Étape 3 — Rends ton Sheet public en lecture : Partage → "Toute personne avec le lien peut voir".

Format du Sheet attendu : colonnes dans l'ordre — Nom, Handle, Plateforme, Abonnés, Niche, Méthode, Statut, Date, Notes
